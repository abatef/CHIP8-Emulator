## Building with CMake
```
$ sudo apt-get install cmake libsdl2-dev
$ git clone https://github.com/abatef/CHIP8-Emulator.git
$ cd CHIP8-Emulator
$ cmake CMakeLists.txt
$ make
```

## Usage

```
CHIP8 ./path/to/rom/file
```
